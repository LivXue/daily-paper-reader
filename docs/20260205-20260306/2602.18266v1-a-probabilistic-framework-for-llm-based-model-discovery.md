## 摘要
从观测数据中自动发现机理性模拟器模型的方法，为加速科学进步提供了一条充满前景的路径。此类方法通常采用代理式迭代工作流的形式，通过模仿人类的发现过程，反复提出并修正候选模型。然而，现有的基于大语言模型（LLM）的方法通常通过手工设计的启发式程序来实现此类工作流，缺乏明确的概率表述。我们将模型发现重新定义为概率推理，即从能够解释数据的机理性模型的未知分布中进行采样。这一视角提供了一种统一的方法，在单一推理框架内对模型的提出、细化和选择进行推理。作为这一观点的具体实例化，我们引入了 ModelSMC，这是一种基于序贯蒙特卡洛（Sequential Monte Carlo）采样的算法。ModelSMC 将候选模型表示为粒子，由 LLM 迭代地提出和细化，并使用基于似然的标准进行加权。在真实世界科学系统上的实验表明，这种表述能够发现具有可解释机制的模型，并改进了后验预测检查。更广泛地说，这一视角为理解和开发基于 LLM 的模型发现方法提供了一个概率视角。

## Abstract
Automated methods for discovering mechanistic simulator models from observational data offer a promising path toward accelerating scientific progress. Such methods often take the form of agentic-style iterative workflows that repeatedly propose and revise candidate models by imitating human discovery processes. However, existing LLM-based approaches typically implement such workflows via hand-crafted heuristic procedures, without an explicit probabilistic formulation. We recast model discovery as probabilistic inference, i.e., as sampling from an unknown distribution over mechanistic models capable of explaining the data. This perspective provides a unified way to reason about model proposal, refinement, and selection within a single inference framework. As a concrete instantiation of this view, we introduce ModelSMC, an algorithm based on Sequential Monte Carlo sampling. ModelSMC represents candidate models as particles which are iteratively proposed and refined by an LLM, and weighted using likelihood-based criteria. Experiments on real-world scientific systems illustrate that this formulation discovers models with interpretable mechanisms and improves posterior predictive checks. More broadly, this perspective provides a probabilistic lens for understanding and developing LLM-based approaches to model discovery.

*   **核心问题**：现有的 LLM 驱动方法无法明确其推理的目标分布，导致难以进行理论分析、难以保证收敛性，且无法系统地处理模型的不确定性。
*   **整体含义**：本论文将“模型发现”重新定义为一个**概率推理问题**，即从后验分布 $p(m|x_o)$ 中采样，其中 $m$ 是可执行的程序代码，$x_o$ 是观测数据。

### 2. 方法论：ModelSMC
*   **核心思想**：引入**序贯蒙特卡洛（Sequential Monte Carlo, SMC）**框架，将候选模型视为“粒子”，利用 LLM 作为提议分布（Proposal Distribution）来迭代地改进这些粒子。
*   **关键技术细节**：
    *   **粒子表示**：每个粒子包含模型代码、权重以及由 LLM 生成的上下文反馈。
    *   **算法流程**：
        1.  **重采样（Resampling）**：根据当前粒子的似然权重进行筛选，保留表现好的模型。
        2.  **传播（Propagation）**：以概率 $\alpha$ 克隆原模型，以 $1-\alpha$ 的概率调用 LLM。LLM 根据祖先代码、性能反馈和任务提示词生成新代码。
        3.  **加权（Weighting）**：计算观测数据在当前模型下的边际似然。
    *   **似然估计（NLE-PFN）**：由于科学模拟器通常不可微且难以直接计算似然，作者使用了基于 **TabPFN** 的神经似然估计方法。这种方法无需针对每个新模型重新训练，实现了高效的“无训练”似然评估。
    *   **参数边际化**：在评估模型 $m$ 时，通过蒙特卡洛积分对模型参数 $\theta$ 进行边际化处理，从而专注于评估模型结构的优劣，而非单一参数点的拟合。

### 3. 实验设计
*   **实验场景**：
    1.  **受控验证**：在有限的高斯混合模型（GMM）空间内验证 SMC 推理的有效性（不使用 LLM 生成）。

*   **底层模型**：主要调用了 **GPT-5-mini**（通过 DSPy 框架）作为代码生成和反馈生成的引擎。

### 5. 实验数量与充分性
*   **实验规模**：每个任务均进行了 **10 个独立随机种子**的重复实验，并提供了 90% 的置信区间。
*   **充分性**：实验涵盖了从简单合成数据到复杂真实世界（R 语言编写的药理模型、神经科学数据）的跨度。

### 6. 主要结论与发现
*   **性能优异**：ModelSMC 在发现准确且具有机制解释性的模型方面优于基准方法，尤其在处理真实世界复杂系统时表现稳健。
*   **后验集中**：实验观察到粒子群体会迅速向高似然区域集中，能够发现与已知生物物理学原理一致的机制（如特定的离子通道动力学）。
*   **理论一致性**：论文在理论上证明了 ModelSMC 在理想条件下是**渐近一致**的，即随着粒子数增加，它能收敛到真实的模型后验分布。

### 7. 优点
*   **理论框架统一**：将零散的 LLM 代理技巧统一到了严谨的概率推理框架下。
*   **不确定性量化**：不同于只给出一个“最优”模型的传统方法，ModelSMC 提供了一个模型群体，可以揭示模型空间中的非唯一性（即多个不同机制可能同样解释数据）。
*   **跨语言支持**：由于其概率框架的通用性，可以轻松处理 Python 之外的语言（如实验中的 R 语言）。

### 8. 不足与局限
*   **计算成本高**：每个候选模型都需要进行多次模拟以估计似然，这在模拟器运行缓慢时会成为瓶颈。
*   **依赖 LLM 提议质量**：如果 LLM 无法提出合理的模型结构（即提议分布的支撑集不足），算法将无法收敛到真实的后验。
*   **代理模型偏差**：似然估计依赖于 TabPFN 等代理模型，若代理模型在特定领域失效，会直接导致粒子权重计算错误。
*   **缺乏语义几何**：目前将代码视为离散粒子，尚未利用代码之间的语义相似性来构建更高效的提议分布。
